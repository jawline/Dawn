The structure of a RAM disk is as follows

RAM Disk Header - A single RAM disk header at the start of the disk containing the size in bytes of the disk

Directorys Chunk Header - a integer with the number of directory headers

The directory headers
	-Name [128 characters]

	-number of directorys
	-dentrys

	-number of entrys
	-fentrys


The file headers
	-Name [128 characters]

	-start
	-size

The file data
000000000000001000000000000000010000000000000000111000000000 etc
